# ✊📄✂️ Rock Paper Scissors Game

## 📌 Description
This is a Rock Paper Scissors game developed using Python as part of the **CodeSoft Python Programming Internship**. The user plays against the computer, and the winner is decided based on the game rules.

## ✨ Features
- User vs Computer gameplay
- Random computer selection
- Win/Lose/Tie detection
- Score tracking
- Play multiple rounds

## 🛠️ Technologies Used
- Python 3
- random module

## ▶️ How to Run
1. Clone this repository.
2. Run the Python file.
3. Enter Rock, Paper, or Scissors.
4. View the result and play again if desired.

## 📚 Concepts Used
- Random Module
- Loops
- Conditional Statements
- User Input
